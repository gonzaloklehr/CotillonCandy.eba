# CotillonCandy.eba
Landing Page de Cotillón para presentar el trabajo con la siguiente estructura: 

1. Estructura HTML: header, nav, main, section, footer
2. Formulario de Contacto: Formulario funcional: Nombre, Correo Electronico y Mensaje (Formspree) para el envió de datos.

3. CSS: Archivo styles.css debería incluir:
	* Header, Footer y lista de navegación.
	* Fuentes de Google Fonts
	* Background (color, imagen, degradado).

4. Diseño Responsivo Flexbox y Grid
	+ Sección Productos: Cards de forma respoonsive utilizando Flexbox.
	+ Sección Reseñas: Utilizando Grid, con una distribución lógica y estética. 
	+ Sección Contacto: Responsive mediante Media Queries para distintos tipos de pantalla.

5. Contenido Multimedia y Navegación
	- Multimedia
	- Lista de navegación: Lista desordenada con navegación interna (Inicio, Productos, Contacto)
